# Skies Beyond the Star

This is the novel outline tree for *Skies Beyond the Star*. Models with good creativity are welcomed. **Manuscript writing must be in English** (ruling 0001); design and canon documents may be in any language (default Chinese).

> Image caveat: except Anna Freedman and Lia, images are not guaranteed to depict Archeon — see `50-assets/ASSET_INDEX.md` and distinguish carefully.

## 怎么读这个仓库

**入口：[`CANON_MAP.md`](CANON_MAP.md)**（权威层级 + 加载协议 + 文件地图）。拼写以 [`GLOSSARY.md`](GLOSSARY.md) 为准。

| 目录 | 内容 |
|---|---|
| `00-meta/` | 流程、裁决、checklists、仓库总规划（repo-plan-v0.2） |
| `10-canon/` | 世界圣经（权威最高，低号覆盖高号） |
| `20-craft/` | 写作宪法（童年篇宪法与设计脊柱） |
| `30-design/` | 结构工程（arcs / modules / ledgers / 待决问题） |
| `40-manuscript/` | 正文（英文） |
| `50-assets/` | 视觉资产（逐图 canon 地位见 ASSET_INDEX） |
| `60-taste/` | 评点层——品味语料：rulings / paired-samples / reveries / 朱批 |
| `90-archive/` | 反面教材库（specimen 禁止作为事实来源，必须与批注同读） |
