### Anna-Lia · 六年间距（14-19岁 baseline）

Anna 14岁飞走之后，她回Atheria的频率是每三四周几天。Lia在谷底生活。Anna在崖壁上层的鹰巢。物理距离本来就不近——从鹰巢到谷底要坐升降笼，走不同的通道层。Anna回来的路线是空港→鹰巢→谷底维护点→回家。她不需要经过Lia住的区域。

但Atheria三万人的社区里完全零交集不现实。

更准确的baseline：

```
14岁（Lia 8岁）    Anna飞走。回来时在栈道上偶尔远远看到Lia——
                    一个走得慢的小孩在谷底。不主动走过去。

15-16岁（Lia 9-10岁） 节日集会上在人群里点过头。市场日在同一个摊子前
                    擦肩过。"认识但没有理由说话"的距离。

17岁（Lia 11岁）    Miriam某次提过一句"Lia那孩子最近好些了"。
                    Anna没有追问。Miriam也没有多说。
                    两个女人之间有一个沉默的共识：不深入这个话题。

18岁（Lia 12岁）    Anna某次回Atheria做崖壁工坊的维护检修，
                    在谷底common house吃午饭时，Lia坐在长桌的另一端。
                    两人之间隔了五六个人。Anna看到Lia在咳——
                    那种短促的、controlled的咳。灰肺的咳。
                    Anna的手指在碗沿上停了一下。然后继续吃。
                    她们没有交谈。

19岁（Lia 13岁）    Arc 2开始。baseline温度：
                    同一个峡谷住了六年，认识但没有真正交谈过。
                    Anna知道Lia的名字、知道她走路慢、知道她还活着。
                    Lia知道Anna是那个穿红夹克的飞行员。
                    两人之间有一条从未被说出口的因果线。
```

这个温度刚好够冷——Arc 2首次named出场时有tension。又够暖——两人之间不存在敌意。只有Anna单方面的guilt和distance。

; [Lia那侧的六年]
; Lia在这六年里长大了。从7岁到13岁。她的肋骨早已愈合。
; 灰肺是慢性的——不会好也不会突然恶化（直到恶化）。
; 她在Atheria谷底的生活是：走得慢、看得多、被工坊拒绝、
; 发展出"所有东西最后都是杠杆"的思维方式。
; 她对Anna的态度不是怨恨——更接近一种watchful的neutral。
; 她知道红夹克。她知道那个人每三四周回来一次。
; 她不主动走过去。Anna不走过来。
; 这个对称的距离是两个人各自选择的结果——
; Anna选的是guilt-driven avoidance，Lia选的是什么？Open question。
; （Ep65-67之间resolve：Lia知道。一直知道。距离是informed choice。）
