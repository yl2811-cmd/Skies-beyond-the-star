**【性格——用行为说话】**

Kassia是那种走进房间不看任何人但知道房间里所有人在什么位置的人。她飞Razor的方式和Anna飞阵风号完全不同——Anna是"听风然后和它合作"，Kassia是"读风然后比它快"。skiff和hauler本来就是不同的飞行器，但差异不在机器——在人。两种方式都能活着落地，但代表着完全不同的和世界相处的哲学。

她的审视感不是敌意——是一种永远绷着的习惯。她掂量风险、掂量距离、掂量人。她很少先开口说话，但说话的时候每个字都是称过重量的。

; [★ 飞行方式 / 精确度 / 虚无主义：同一棵树的不同枝]
; Kassia的世界观核心：宇宙不会接住你。父母消失了，世界继续转。Federation沉默500年。
; 从这个前提出发的心理推论：如果世界不会接住你，你只能比世界快。
;
; Anna"听风然后和它合作"的前提是信任——风会给你信息，你和它是同一个系统的一部分。
; Edmund教的"手放上去，等，让东西告诉你它是什么"是同样的逻辑——对物质世界的basic trust。
; Kassia没有这个trust。"读风然后比它快"的前提是：风不会配合你，它只服从物理定律，
; 你要么比它先一步，要么你死。她不是在和风对话——她在跑赢风。
;
; 精确度（0.3mm偏心不接受）不是工匠的perfection——是对不确定性的零容忍。
; tolerance range = 把一部分结果交给概率 = 信任"世界在大部分时间运转正常"。
; Kassia不信任大部分时间。父母在一次正常航线上消失了——概率事件，不可预测，没有预警。
; 所以：把能控制的范围内所有变量压到极致。under-correct可能死，over-correct只浪费能量。
; 精确度是Kassia的armor，不是她的美德。
;
; 飞行对Anna是connection（到达→deliver→和人交换）。
; 飞行对Kassia是disconnection（离开→高度→一个人）。
; 两种飞行方式在同一片天空共存 = 全书最好的哲学对话之一。不需要任何台词。
;
; 因果链总览：
;   父母消失（世界不接住人）→ 宇宙不在乎（CI蚀刻板cosmic isolation）→ 不信任世界自我修正
;     ├→ 飞行：比风快。预判。抢先correction。永远不放松
;     ├→ 精确度：0.3mm不接受。不把结果交给概率
;     ├→ 人际：永远绷着。不先开口。说话称重量
;     └→ solo flight：没有destination。above it all。回避density
;   全部是同一个root的不同枝——"如果宇宙不接你，你就永远不能松手。"
;
; Anna/Lia的erosion路径（通过飞行precision渗透）：
;   Ep38 slot — Kassia第一次看到"听风"有效（不控制也能活）
;   Ep57 — 试了一次"不修正"→没死→记住了
;   Ep60 — precision保护了Anna→技能从armor变成care
;   Ep62 — correction angle缩小→多等了一拍→trust +0.01
;   Ep69 — 手放在墙上。不是在读什么、不是在预判什么。什么都没做。只是放在那里。
;          Kassia一辈子用手做的事是控制——操纵杆、校准环、绞盘。手是工具。手是armor。
;          触墙的手什么都没做。这是Kassia版本的"手放上去，等"。
;          她不知道。但手知道。

**（Kassia在童年章节不出现。她是Arc 1之后的关键角色，在Arc 3-4有重大发展——通过Anna可见的行为和physical evidence呈现。）**
