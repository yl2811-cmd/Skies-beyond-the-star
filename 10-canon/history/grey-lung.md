### 3.3b Sky Fire的遗传印记——灰肺（Grey Lung）

Archeon距参宿四约10光年。超新星爆发的高能辐射——gamma射线、宇宙射线——穿过大气层时被部分屏蔽，但掩体外的剂量是致命级的。掩体内survival expected，但辐射仍穿透了部分屏蔽层——不致死，足以造成DNA损伤。

CI准备了genetic repair kits。但那是25世纪技术，需要CI级医疗设施支持。Sky Fire同时摧毁了Axiom和大部分CI基础设施。基因修复工具在最需要的时候失去了维护它们的基础。

因果链：

```
第一代掩体幸存者（Atwood等）→ 辐射诱导DNA损伤
    ↓
CI genetic repair kits最初几十年内处理大部分急性损伤
    ↓
CI医疗系统退化（零件耗尽、知识断层）→ 修复越来越不完整
    ↓
低外显率突变在种群中积累
  × 500年、几十代人
  × 小种群基因瓶颈效应放大
  × 不完美修复的残余
    ↓
特定慢性易感性表现型——"灰肺"是其中之一
```

**灰肺的医学机制：** 肺泡表面活性物质的合成通路存在微妙缺陷。肺泡不能完全有效地扩张，气体交换效率比正常低15-20%。不致命——但走快了会喘、剧烈活动受限、容易因肺部感染或外伤（如肋骨骨折导致的肺挫伤）产生比正常人更严重的后果。

**种群分布：** 不是"一个人的遗传病"——Archeon整个种群都携带Sky Fire遗传印记。大部分人携带损伤但不表现。少数人碰到了不利的等位基因组合，表达出来了。Lia是其中之一。某些家庭出这个问题的概率更高——不是单基因遗传，是多个受损位点的组合效应。

**Archeon人自己的理解：** 他们没有"基因组学"的语言。他们知道"有些孩子生下来肺不好"、"有些家庭出这个问题的多一些"。长辈们叫这个**"灰肺"（Grey Lung）**。有经验的医者知道Ember Crown提取物能缓解——但不知道为什么。这是500年知识断层的直接后果：CI知道原因，但那些知识随CI一起沉没了。

; [叙事层功能]
; Lia的身体问题不是个人的"坏运气"——是整颗星球的伤疤在一个七岁孩子身上的具体表现。
; Anna用扳手伤害了Lia，但Lia的肺从出生前就已经被一颗521年前爆炸的恒星伤害了。
; 两层因果叠加：Anna的guilt获得一个更大的框架——她伤了一个已经被伤过的人。
; Ember Crown药用提取物恰好是Archeon目前能找到的、对灰肺最有效的缓解药物
; ——Skyleaf的不可替代性和Lia的生存直接焊在一起。

; [硬科幻校验]
; ✓ 参宿四距离10光年 → 辐射剂量：掩体外致命，掩体内亚致死但造成DNA损伤
; ✓ CI genetic repair kits是25世纪技术 → 需要CI级设施维护
; ✓ Sky Fire摧毁基础设施 → 修复工具在最需要时失去支撑
; ✓ 500年小种群 → 基因瓶颈放大低外显率突变
; ✓ 肺泡表面活性物质缺陷 → 符合"不致命但functional limitation"的叙事需求
; ✓ Ember Crown的药用价值 → 不需要解释为什么有效（Archeon人不知道机制）
;   只需要经验性知识"这个花的提取物让灰肺的人呼吸好一些"

机器人工作群滞后CI约一个阶段退化——失去高层协调后退入本地循环，逐个停机。每周多干一点，每月多修一台，每年少一台机器人。不是突然断裂——是漫长的交接。

**为什么没人来（Long Silence）：** 不是被遗忘，不是被物理切断。四层叠加：

**第一层：信息断裂。** 参宿四~2500年坍缩。光速传播，地球要到~3010年才亲眼看到超新星。在那之前Federation只能通过间接证据（信使失联、航线异常辐射）推测出了问题。从发现异常到确认参宿四坍缩，可能耗去十到二十年——不是愚蠢，是520光年距离下的信息物理学。

**第二层：物理与技术瓶颈。** 走廊辐射风险（见§3.3c分级表）在超新星后10-50年内使曲速航行近乎自杀。50-200年间可行但代价极高。同时曲速技术本身卡在类事件视界墙上——100-150c的速度意味着520光年单程3.5-5年。每一任决策者都可以对自己说"等引擎再快一倍就值得了"。

**第三层：注意力经济。** Federation没有资源瓶颈，但有注意力瓶颈。苏醒者只占总人口的15-25%，其中真正关心星际事务的也许十分之一。这些人的注意力被核心圈殖民地的日常运营（Phase 3的CI渐退需要大量人类参与和决策）、新候选行星的评估论证、以及26世纪后意识上传浪潮引发的社会形态分化完全占据。Archeon在520光年外——不是不重要，是太远了，没有人天天想着。每次讨论结论都是"再评估""再等等"。不是冷酷的Triage计算，而是一种更隐蔽的东西：后稀缺社会的注意力像水一样流向阻力最小的方向，远端的、代价高的、结果不确定的项目自然沉到议程底部。

**第四层：道德钝化。** 第一代决策者知道Archeon上有人。第二代决策者知道上一代决定不去。第三代决策者知道"那边可能已经没人了"。每一代都有合理借口。累积起来就是五百年遗弃。Federation没有开会正式决定"放弃Archeon"——他们只是不断地不做决定，直到不做决定本身变成了决定。~2900年第二代曲速架构突破后技术借口彻底消失——再也没有"我们做不到"这块遮羞布。但此时道德欠债反而最重。
