### 2.7b 交通——浮力是Archeon的DNA

**核心原则：没有车辆，铁轨极罕见，人主要在近处活动。所有需要跨较远距离的系统尽可能**借浮力承重、用最小能源引导方向**。同样的电，Archeon系统能用十倍远。**

**① Skybridge步行** — 90%的日常短距离出行。三维网络：同层水平 + 不同层斜桥。materials见上§2.7。功能overload：交通 + 晾衣 + 摆摊 + 社交。多数人一辈子的日常活动半径 = 住处方圆500米之内。

**② Gondola索道** — 中距离 + 垂直高差。Frontier City 12-20站点、30-40条line。木质外壳 + 黄铜件 + 陶瓦顶的小gondola，4-8人。动力：聚变堆电驱绞盘；最大line还用重力循环（一上一下配对，几乎不耗能）；主干line的gondola内部有小hydrogen囊减重30-40%。速度15-25 km/h，慢、从容、社交时间。缆绳过站点滚轮的"挞-挞-挞"是Frontier的signature声。**Atheria、Iris Hollow、其他主要settlement都有小规模索道**。

**③ ★ Float Rail / 浮轨** — 公共轻轨的Archeon版本。basalt fiber单轨架在支柱之间，车厢顶部椭圆hydrogen囊使车厢接近中性浮力（微微偏重不会飘走）。**轨道不承重——只从下方限制车厢不飘上去、从两侧限制不被风吹偏**。推进用小电动涡轮（几百瓦），刹车用蒸汽反向喷口（蒸汽回收供暖）。速度25-40 km/h——Frontier内最快的公共交通。视觉signature：一串墨绿色大蚕蛹形车厢在空中悬挂单轨上缓缓滑过。**Frontier只有2-3条Float Rail line**（连接核心pylon群和外缘聚变堆/住宅区）；Iris Hollow和Amber Reach的缓坡地形也适合。Atheria不适合（地形不允许直线）。

**④ 气动管网络** — 历史上巴黎伦敦都有过的蒸汽压力邮管，Archeon强化版。铜管（CI遗产）贯穿pylon和terrace核心，蒸汽压力送胶囊，速度30-50 km/h——比所有载人交通都快。用于：信、小件急用零件、Guild manifest、紧急医疗品。**不送大件不送食物**。声音signature：背景音里每隔几分钟有一声从墙里传出的远处"嗖——"。Festival期间管网经常backup——认识管网操作员成为Frontier特有的人情货币。

**⑤ Cargo Chute / 货物滑道** — 封闭basalt fiber大管内衬heartwood滑面 + 蒸汽制动阀，从高sky pier直接滑到地面仓库。**只向下，重力驱动**。Festival期间有人把灯笼放进chute——暖光从150米高一路下滑，外面观察窗像一条发光线。

**⑥ Sky-Lighter / 浮渡艇** — Archeon对"私人车辆"问题的答案。2-3米长、1人（或1人+小行李），小hydrogen囊给中性浮力，脚踏杠杆 + 小蒸汽冲压罐推进，把杆+小帆控向。速度8-15 km/h。优势是point-to-point不走预定路线。**中产象征**——专业人士（guild master出诊）、富裕家庭、年轻人约会（经典scene：一对年轻人各划一艘sky-lighter在低空飘着聊天）。Anna没有。

**⑦ Lift / 垂直升降笼** — 水力/蒸汽，嵌在pylon内部或贴崖壁外侧。慢（20-30秒一层）但稳定。Atheria的崖壁升降笼 = 城市版lift的峡谷亲戚。

**⑧ Pulley-basket / 辅助** — 邻居之间的小物件，挂绳挂篮，手拉。

**⑨ 窄轨铁路** — **极度niche**。只出现在主skyport底层pier的货栈内部，窄轨（约1米轨距），手推或thorn-goat拉小平板车。**绝不用于载人，绝不延伸到货栈以外**。

**Frontier傍晚的天空分层**（傍晚抬头看）：近景路灯市集 → 中景skybridge网络 → 稍远几只sky-lighter像蒲公英漂 → 更远一串gondola暖黄萤火沿斜线移动 → 最远一辆Float Rail车厢（像发光大蚕蛹）缓缓穿过两个pylon之间 → 最高pylon顶signal lamp + 停靠的飞艇 → 天幕Sky Fire's Remnant玫瑰色光河。没有一样是地球的。

**其他settlement交通速查：**
- Atheria：升降笼（蒸汽/水力驱动）+ 坡面缆车gondola（水力counterweight驱动）+ 坡面短距离空桥（同侧工坊阶地之间，几十到一百米级别）+ 谷底氢电池窄轨平板车（沿Ishkar河岸，运货载人）+ 谷底石拱桥和木桥横跨Ishkar河（几十米跨度）。无跨谷skybridge（谷宽2km+，超出任何蒸汽朋克技术跨度）。无Float Rail（地形）。无sky-lighter日常（谷中乱流）。气动管网络小（十几站）。从一侧坡面到另一侧：下到谷底→过桥→爬另一侧之字路。或者：playful的方式是坐滑翔架从坡面中段滑到对面落在另一侧阶地上。
- Iris Hollow：Float Rail 1-2条（沿丘陵坡线）。sky-lighter流行（风稳空间开阔，年轻人有sky-lighter赛事）。gondola较少。
- Amber Reach：Float Rail主导（连接农庄聚落到镇中心）。气动管传递农技信息。无gondola。
- Riverbend：水渠 + 步行 + pulley-basket为主。一条气动管（skyport↔smoking racks，送鱼货分配单）。无gondola无Float Rail。
- Skyleaf、其他小settlement：步行 + pulley。偶尔一条短gondola。其他系统能源不够。
