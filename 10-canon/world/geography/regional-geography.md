### 关键地理单元

| 地理单元 | 特征 | 尺度 |
|---|---|---|
| **Main Basin** | 温带宜居区，大部分人口和agriculture | ~800 km E-W，略椭圆 |
| **Sea** | 大陆西侧和西北侧的海洋 | Iris Hollow面海，Driftmoor/Tidemark/Shellharbor沿海 |
| **Orun–Marneth Arc** | 盆地南缘到东缘的连续造山脊系，两段在东南角以~30°浅夹角相接=**开放浅弧，非围栏**。盆地朝北/东北敞开。 | 南段~800km E-W + 东段斜向N |
| **— Orun Plateau（西/南段）** | 逆冲断层抬升的basalt台地。北面35° escarpment 3km落差（Titan's Staircase），南面缓降高原2-3km。西端在海岸处降至800m-1.2km（The Cape海蚀角）。地质不活跃。 | 东西延伸~800 km |
| **— Marneth Peak / Crown（东南转角）** | 全弧抬升最猛、冰川化最强的顶冠，约6000-7000m，整条山脉链的最高culmination，常年冰雪。从盆地朝Orun望去，越过高原东端升起的白色雪冠（南偏东vista）。 | 最高~7000m |
| **— Marneth Range（东/北段，旧称Serpent's Tooth）** | 自东南转角斜向北延。**classic层叠巨型山脉(非平直墙、非Orun台地阶梯)：脊平均海拔5-6km、最高峰群近7000m、高地山脚~4000m≈雪线；盆地侧约30km水平渐次抬升(雪线下山脚植被+多条河流下切，雪线上山峰层层叠高到峰顶)。全crest皆在雪线以上、最低处>5km、无低于升限的鞍部。** 高于飞艇升限（见§2.10b）→翻越物理不可能。**北翼坐Skyleaf撞击坑。** | 脊5-6km，峰~7000m |
| **Skyleaf 撞击坑** | Marneth北翼，几千米级撞击坑，径数十km；雪峰环壁、坑底绿草原+多湖+向心溪、矿物/地热富集。唯一缺口=Kestrel's Gate。 | 坑径~数十km |
| **Kestrel's Gate** | **Skyleaf撞击坑壁东南向唯一缺口（SE breach）**；通断=季节性盛行风经Venturi加速→封季整季gale（对外切断）/开季温和；昼夜热差降为fine structure（隼门暴风成因）。 | 缺口窄，两侧坑壁数千m |
| **Grand Canyon** | Orun Plateau被Ishkar河切穿形成的峡谷 | Riverbend在峡谷内 |
| **East Mesas** | Marneth斜脊以东、Zone 4野化草原深处的低矮台地群——CI原规划的terraform扩展区遗存。**东方generic vista锚点（非Skyleaf专属）；也是近东↔深东的视觉边界。** | 东方地平线 |
| **East Wildgrass（东方野化草海）** | Marneth东侧、Ironvein东向加深的同一套火山-铁地质在草海里展开。CI缓冲带自维持植被=不需terraform天然可居。开放草海（直线航路）。Cinderlode + 一圈草海散点的所在。 | Marneth以东广阔 |
| **Cinderlode 火山裂谷** | Cinderlode锚定的母地貌：草海里一道开放的年轻火山裂谷，地面冒蒸汽、蓝绿热水池阶梯爬坡、travertine沉华阶泉、一面玉色矿物湖；红黑BIF铁崖+磁异常（罗盘失灵）、硅化先锋林（石树）为伴生特征。**开放·大天空——与Skyleaf封闭雪环碗成镜像。**（备选flavor=六棱玄柱阵+lava-tube大城。） | 裂谷+矿物湖锚定80k城 |
| **沉溺峡湾 / 北境** | 北/西北海岸的rias/fjords（Driftmoor）+ 跌落高原阶梯。可探索可居。 | 北境 |
| **岛山·桌山 / 地堑谷** | 盆地内桌山（中部、东北缓丘；Crestline）、地堑断陷谷（Gearstead）；另有内陆矿物晶斑。 | 盆地内 |
| **Great Ochre Desert** | Orun以南的赭红色沙漠。内含**盐脊带**（盐穹/盐冰川）、**低语晶原**（结晶古湖床）、**锈带崖**（红黑BIF+磁异常） | ~400 km到南海岸 |
| **熔管网 / 玄柱台地** | 盆地古熔岩流抽空形成的公里级地下管道 + 六棱玄武岩柱阵。地下"道路"与庇护所（部分settlement建于管内） | 散布Basin中南部 |
| **倒置雾海盆（"白锅"）** | 盆地封闭深盆里持续数日的逆温雾海 | Basin内数处 |
| **双月潮海岸** | Azure Main海岸受赭/银双月拍频复合潮支配（巨型潮间带+潮涌+双月潮历） | 西/西北海岸 |
| **Axiom Scar** | Axiom号坠毁残骸带，在Desert内 | ~100 km debris field |
| **The Cape（海角）** | Orun西端海岸终止点。Escarpment降至800-1.2km，海蚀台地+拱门+海蚀柱。常年海雾。稳定updraft。Orun最低翻越点。 | Arc 4南行路线关键节点 |
