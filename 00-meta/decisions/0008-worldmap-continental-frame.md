# Decision 0008 · 大陆级世界地图框架裁定（尺度勘误 / 冰盖 / Ishkar 水系 / 森林覆盖公式）

**裁决人：** 作者（2026-06-10，工作对话 + 手绘大陆草图）

本裁决以作者手绘大陆草图为准（草图待作为文件入库 `50-assets/maps/`——本 PR 未含二进制，入库前以本裁决文字为权威），统一大陆级空间事实，并销三处库内尺度矛盾。

## 1. 大陆尺度（勘误，作者确认）

- Main Basin 东西 **~800 km**（`regional-geography.md` 旧值 "200+ km" 为录入错误，作废）。
- 大陆主体东西 **~2500 km**，南北跨度 **3000 km+**：主体横向拉长，北侧以荒地颈部延伸直连北极冰盖。旧 "约2000km级" 作废，全库统一 2500。
- 灰圈 = 旧 `§1.2` ASCII 取景框的范围，仅为西部特写；正式地图保留为图例元素。

## 2. 冰盖（部分消融，物理理由锁定）

terraform Phase A 只做到极地冰盖**大幅退缩**（南缘海冰消融、Azure Main 开放），**极地核心冰盖保留**。理由（作者原话方向）：聚变供能的温室气体注入是渐进的，无法一次性消融全部极地冰——大陆北缘直连北极冰盖这一事实因此成立。`atmosphere-origin-and-terraform.md` 旧 "极地冰盖消融→开放海洋" 收窄为此义。

## 3. Ishkar 水系（新结构，proposed → 见 hydrology-ishkar.md）

basin **无大型湖泊**（零星小湖可有，不命名、不入图）。水以河渠为主：

- **北支**：自北缘/山脉下行，**部分径流向海**（西/西北 Azure Main）。
- **南支**：自 Orun 北壁下行。
- 南北两支于盆地汇成主干 → 盆地内河流因此多被泛称 **Ishkar**（解释了 Atheria〔东北谷〕与 Riverbend〔南 Orun 峡谷〕同名却异位：同一水系的不同支）。

**待裁**：Riverbend 所在主流 = 南支 / 主干？（hydrology-ishkar.md 暂挂 proposed）

## 4. 森林覆盖（直接套 3-5x 公式，不做真实降水线推演——作者指令）

模型：**定殖锋面距离 = 地球基准锋速 × 加速倍率 × 时间**。常量显式可调：

| 参数 | 取值 | 来源 |
|---|---|---|
| 基准锋速 | 0.4 km/yr | 地球后冰期乔木迁移 0.1–0.5，工程化先锋种取中高段 |
| 加速倍率 | 3–5 | canon 加速生长开关（ecology §1.4b） |
| 时间 | 521 yr | Sky Fire 后（年历锚点） |

**结果：自然 halo ≈ 600–1040 km。** 叠加西部播种核心（~800 km basin）→ 植被触及 ~1600 km，约占大陆东西 65%。

落账三层：
1. **连续密林** = basin 核心 + Orun 迎风面（播种核心）。
2. **自维持草原疏林** = 核心外推 600–1040 km halo，向东渐稀，覆盖 Zone 4 东方草海。
3. **无植被** = halo 之外（深东 East Mesas 以东、Great Ochre Desert、北方荒地）。

此结论写入 `ecology.md` Zone 1/4；公式与常量留痕于此，供后续 session 重算调参。
