# prose-gate · 散文过审单

> 每段 prose 过 PR 前逐条自检。任一条不过 = 修复后重审；每处修复的前后两版按收割纪律进 paired-samples。

## POV / focus（宪法 §一）

- [ ] 镜头没有切走：没有任何 Anna 不在场的场景、没有"与此同时 X 在想……"。
- [ ] 场景引擎是"Anna 在感知世界"，不是"Anna 和某人的关系推进了一步"。
- [ ] 命名配角停留在感知层（感官签名/复现 beat），没有内心戏、没有支线升格。

## Edmund / Miriam（宪法 §三、§四）

- [ ] Edmund 是 world 不是 flashback：没有内心闪回段落；他存在于物（logbook、工具余温、红夹克、三声部）。
- [ ] Edmund 的对话回应是 Pattern A（shared delight）或 Pattern B（荒诞幽默）；**没有 Pattern C（把 Anna 当故障物诊断）**。对话每次都短、每次不一样。
- [ ] Miriam 保持背景常量暖点密度（茶/新杯子/七字民谚级），不解释、不抢镜。

## body-as-narrator（宪法 §3.2）

- [ ] 悲伤最高点扣住内心：只报告身体动作，没有"她感到悲伤"式情绪报告。
- [ ] 刀在缺席与余光（少掉的声音、只剩一只的杯子），不在被描写的悲伤。

## exposition 禁令（宪法 §五 + 年历部署红线）

- [ ] 世界观通过 Anna 的身体感知投放，没有 exposition 式投放。
- [ ] 季节/年历信息没有以陈述句进入 prose（没有"凉季到了"）；季节只被闻到、听到、踩到、穿到；唯一允许的语言形态是民谚短句。
- [ ] 红线词检查：当前 episode 是否在全局信息禁令期内（"参宿四"Ep1 全集禁；"Sky Fire" Ep1–4 禁；天文坐标/星图 Ep1 禁——见 30-design/arcs/intro-arc.md §一）。

## 成功标准

- [ ] 读完的感觉是"我在 Archeon 里活了一天"，不是"我读完了一章情节"。
