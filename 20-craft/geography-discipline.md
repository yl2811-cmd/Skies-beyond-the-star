[## ★★★ CRITICAL — 地名使用规则（写任何scene前必读）

### 绝对禁令

**不要随便patch地名。**

如果一个scene需要一个settlement作为背景：
1. 从下方列表中挑一个**地理和功能上合适的**，或者
2. 用**"某个outpost"**代替。

错误的地名比没有地名更糟糕——它会破坏我们的空间连贯性，而且一旦写入正文就无法无损撤回。

### 决策流程

```
需要一个settlement作为场景背景？
│
├─ Anna到过吗？（查 §3 Anna航线覆盖）
│   ├─ 是 → 可以用named settlement，查 §2 拿感官签名
│   └─ 否 → 她第一次去。这是有意的plot beat吗？
│       ├─ 是 → 可以用，但必须写"第一次到达"的感官冲击
│       └─ 否 → 换一个她去过的，或者用"某个outpost"
│
├─ 这个settlement的地理位置合理吗？（查 §1 区域地图）
│   ├─ Anna从Atheria飞往Iris Hollow要横穿Basin中段，不会经过Riverbend
│   ├─ Riverbend在南方Orun峡谷里，去那里要从Coppergate方向南下
│   ├─ 从Frontier去Skyleaf要向东北绕Marneth北端、再经Kestrel's Gate入坑（Marneth是冰墙穿不过；不是向南）
│   └─ 沿海settlements（Driftmoor/Tidemark/Shellharbor）不在内陆航线上
│
├─ 这个settlement的功能/气质和scene需求匹配吗？
│   ├─ 需要安静偏远 → 不要用Frontier City
│   ├─ 需要繁忙贸易 → 不要用Saltmarch
│   └─ 需要水/鱼 → Riverbend，不是Atheria
│
└─ 如果以上都不确定 → 用"某个outpost"或"一个中转skyport"
```

### ★★ Ep36 特别注意

Ep36发生在 **Frontier City**，Thread Festival期间。三个beat的物理位置：
- **Kassia招募** → Frontier City docking platform
- **Anna-Lia飞行话题** → Festival期间，具体位置flexible
- **Anna-Lia独处** → 最高pylon顶端悬挑pier末端的木构观测亭（CI骨架+150年人类加建）

### ★ "Basin"是CI/Federation的地图词——Anna那一代人不说这个词

**800×500km的低地盆地从盆地里面看起来是什么样的？** 是开阔的大地——但不是一片均匀的平面。地平线主导视野，但地形在悄悄起伏：田块和颤叶林带、低矮丘陵、浅切河谷、孤立的basalt台地、湿地和草坡，每隔几十公里换一种地貌（详见§1.2 "Main Basin的内部地形分异"）。Atheria峡谷的人走出谷口，看到的是地平线延伸到天际——但那条地平线不是一条直线，是被远处的丘陵脊线、台地剪影、防风林带打断的有节奏的轮廓。Orun高原的北壁确实像一堵横跨天际的墙，但从basin内部看那只是"南边的山脉"——没有人站在Basin里面觉得自己在一个bowl里。你需要从轨道上往下看才能看到Basin的形状。Archeon没有轨道视角。

**命名演化时间线：**
- **CI时代（~2470-2506）：** "Basin Seven"是CI的行政区划代号。CI的语言是测量和分类的语言。
- **Atwood一代（~2506-2560）：** 第一代居民继承了CI的术语。"Basin""Main Basin"在日常用语里还存在——像殖民地的人继续用殖民者给的地名。Atwood本人大概说"the Basin"时还能感觉到这个词的literal meaning——她知道自己住在一个地质盆地里，因为CI告诉过她。
- **第三到第五代（~2580-2650）：** "Basin"开始从"地理描述"变成"习惯叫法"——就像"长安"从"永远太平"变成一个不带字面意义的城市名。
- **521年后（Anna的时代）：** 没有人说"Basin"了。800km宽的低地就是大地。Archeon人用的词是方位和地域性的——"平原""谷地""南边山那头""海那边"。他们可能在非常正式的场合（如古老的地图、Landing Day仪式的历史朗读）听到"Basin Seven"这个词，但不会在日常对话里用，就像21世纪中国人不会日常说"中原"来指河南平原。

**写作规则：**
- **本文档（timelapse）** 作为世界观设计文件，继续使用"Basin""Main Basin"作为meta-level的地理标记词——写手需要一个短标签来快速定位。
- **写实际prose/对话时：** Anna及同代人不说"Basin"。他们说"大地""平原""我们这边""南线""北边那片"——方位性的、具体的、活人的语言。如果某个年长的人偶尔说出"Basin"，那感觉像是在引用旧词——一种学者气或仪式感，不是日常。
- **Intro arc（Atwood时代）可以用"Basin"：** 那一代人确实这么说。Atwood说"break the giants, grow them sideways"的时候，她说的"Basin"还有字面含义。
- **同类generic化的词：** "Settlement"也是。Atwood一代人会explicit说"settlement"来区分"我们建的地方"和"荒野"。521年后，settlement就是"镇""城""家"——没有人在意自己住的地方曾经有个CI分类名。"Colony"更早消失——Archeon人不知道自己是colonist，他们就是住在这里的人。"Terraform"同理——Anna不知道她脚下的草是被设计出来的。

### "某个outpost"的使用指南

当scene不需要specific settlement identity时，用以下模板之一。这些不需要名字。Anna在航线上经过几十个这样的地方。括号内是writer-level的地理定位（meta），引号内是character-level的说法：

- **"一个中转skyport"** — 航线中间的补给点，有引擎维护设施
- **"平原上的一个小outpost"** — （meta: Basin内陆），几百人，草地锚桩泊位，安静辽阔。Anna会说"平原那边的一个小地方"
- **"丘陵区的一个settlement"** — （meta: Basin东侧），靠近Thornfield方向，麦田。Anna会说"东边丘陵那片"
- **"峡谷里的一个小镇"** — Orun系统内某处，水声，阶梯建筑
- **"海边的一个渔村"** — 西海岸Iris Hollow以南方向，盐味，雾
- **"Marneth山脚下的一个前哨"** — Stormperch附近区域，飞行员避风处
