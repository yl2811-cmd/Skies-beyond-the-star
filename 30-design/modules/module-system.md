# Anna童年素材库 — 底层模块

> **引擎定义：** Anna注意到了什么 → 发现一条invisible chain → 世界变大一圈
>
> **Edmund的角色：** 不是老师，是**一起高兴的人**。他最常用的动作是：在旁边等——等她自己发现——然后两个人一起为这个发现高兴。他的回应不是机械的确认，是shared delight的入口——有时候跟着一个问题（"那你觉得为什么？"），有时候是拍一下她后脑勺，有时候是他自己被勾起来的一段回忆，有时候两个人对视一眼同时笑了。**每次回应的形式都不同——见Edmund对话variety强制规则。** Anna童年里"世界变大一圈"的体验不只是认知上的——是情感上的。每一次发现都是和Edmund共享的快乐时刻。
>
> **Miriam的角色：** 教Anna看人，不是看东西。她的感知轴是社会关系，不是物理系统。
>
> **红线规则：**
> - 童年的红 = 铁氧化物红土染料。最常见，最便宜，到处都是。
> - 全部botanical红留给Skyleaf。童年里不出现花田、不出现Ember Crown。
> - 红色在童年里的出现必须**casual到第一次读不会注意**。
> - Edmund关于衣服的解释是**整条红线的承重点**——表面完全practical，深层devastating。
> - 我们第一次读到时的反应应该是"哦，合理"。回想起来时的反应应该是心碎。
>
> **绝对禁令：** 没有任何场景的核心是"Anna修好了一个东西"。她**注意到**东西。

### 两套素材系统的关系

timelapse中Anna童年素材存在两种格式：**散文段落（§1-§5感知通道阶段 + 1.1-1.5场景）** 覆盖3-13岁全跨度的感知发展框架，阶段级粒度（"这个年龄段打开了什么通道"）；**M-module卡片（M-06至M-28，含M-09b和M-18b；M-09 Thread Festival已废弃删除）** 覆盖6-13岁的具体记忆场景，场景级粒度（"那天下午发生了什么"）。两者不是重复——是zoom level不同。1.1-1.5等价于"未编号的M-01到M-05"（写作时间更早采用完整散文），6-7岁overlap（1.5暴雨 vs M-07回声）是同一年龄不同场景。

**写episode时：** 先查§1-§5确认该年龄段通道开放状态和写作约束，再查M-module找具体可部署场景，两者同时参考。

## 模块格式说明

每张卡片包含：
- **年龄** / **感知类型** / **世界半径**
- **场景种子**：一段话描述发生了什么
- **Edmund的moment**：这个场景里Edmund做了什么、他的表情是什么、两个人之间发生了什么。一句话，只写行为和身体语言，不写心理分析。不在场则写"不在场"。**此字段强制模型将Edmund的情感存在作为独立维度读取，防止被感知训练框架淹没。**
- **Invisible Chain**：Anna发现了什么连接（纯joy模块此项为"无"）
- **大人的角色**
- **情感色调**
- **红线状态**：这个模块里红色是否出现，以什么方式
- **下游潜力**：这个记忆在19岁主线中可能被什么moment调用
