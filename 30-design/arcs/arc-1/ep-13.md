## EP13：飞行 · Orun · Grand Canyon（4,000-5,000字）

### 四阶段结构

```
Phase 1: 出发/高原接近（~500字）— basin weather → Orun远景
Phase 2: 迎风面仙境（~1,500字）— updraft飞行 + 地质奇观 ★ wonder peak 1
Phase 3: Grand Canyon reveal（~1,500字）— 侧入峡谷 + two-worlds vista ★ wonder peak 2
Phase 4: 下降/Riverbend初见（~500-1,000字）— 收焦到城镇 → EP14衔接
```

### Phase 1：Basin → Orun接近（~500字）

从EP12结尾衔接。清晨出发。Basin weather平稳。Orun Plateau在地平线上逐渐清晰——一堵行星尺度的墙。

**关键台词**：Anna设定航线——不走最短路径（直飞翻越），而是沿迎风面（北面）向东飞。"We don't go over. We go along. The wall gives us lift if we ask politely."

**tension level: 0/10。** 这是calm before beauty，不是calm before storm。

### Phase 2：迎风面仙境（~1,500字）★ wonder peak 1

**物理基础：** Orun北面（迎风面），35°平均坡度，3km垂直落差。西风携带Azure Main的湿气撞上这面墙，被迫抬升→冷却→凝结→降水。Anna的hauler利用这个orographic uplift——**Invisible Escalator**——沿壁面向东飞行，风自动把飞艇往上抬。

**地质景观（Titan's Staircase）：**
- Basalt层（深黑色）形成垂直台阶的"踏面"
- Shale层（灰褐色/红褐色）形成台阶的"踢面"，风化后退缩
- 层层叠叠的basalt risers上长满翡翠色苔藓（湿气充足）
- 黑色basalt + 绿色苔藓的对比 = 这个世界独有的色彩签名

**瀑布群（Weeping Stone / Spring Sapping）：**
- 地下水从basalt/shale交界面涌出，形成无数瀑布
- 不是Niagara式的轰鸣——是从岩壁上"渗"出来的、宽大的水帘
- 有的是细线般的水流，在阳光下像银丝
- 有的是宽阔的薄膜水帘，飞过时水雾扑到驾驶舱玻璃上
- Anna不躲——这是她预期的。她打开刮水器（如果有的话）或等风吹干

**云海：**
- 迎风面中段（~1.5km高度）以下被云层覆盖——Anna在云层上方飞行
- 偶尔云层裂开一个洞，能看到下方basin的绿色田野，像从飞机窗口看到人间
- 云层上方的世界：黑色basalt阶梯 + 绿色苔藓 + 银色瀑布 + 蓝色天空 + 下方白色云海
- **Studio Ghibli moment：没有对白。没有danger。只有飞行和美。**

**Icing（结冰）：** 作为routine annoyance，不是crisis。某段飞行时挡风玻璃开始结一层薄冰。Anna用手套擦了一下，继续飞。Milo问要不要升高，Anna说"不用，再往东两分钟就过了这段冷区。" 两分钟后确实过了。**Competent pilot处理routine challenge的showcase。**

**这个phase没有对白（或极少）。** Anna和Milo看着窗外，沉默。音乐的空间——如果这是电影，这里是纯配乐+环境音。

### Phase 3：Grand Canyon Reveal（~1,500字）★ wonder peak 2

Anna沿迎风面向东飞行，到达Grand Canyon从Orun侧面切入的位置。

**THE MOMENT：** 峡谷不是从正面出现的——是从**侧面**突然裂开的。Anna沿着一面连续的basalt壁飞行，壁面突然消失了——右边出现一个巨大的豁口。

**视觉sequence（slow reveal）：**

1. 先看到的是**对面的壁**——3km高的垂直石墙，层层叠叠的basalt/shale，在阳光下呈现暖色条纹。距离5km。这不是对面的那座山——这是同一块高原的另一面。中间什么都没有。
2. 然后视线往下——**底部**。3km的垂直落差。Ishkar河在最底部，一条银蓝色的蛇在黑色石头间蜿蜒。从这个高度看，河流细得像一根线。
3. 然后看到峡谷的**纵深**——向北，峡谷开口处能看到basin：绿色、云海、远方的平原，柔和的光。向南，透过峡谷能看到Great Ochre Desert的赭红色——干燥、空旷、热shimmer。**两个世界在一个frame里。**

**"Two worlds in one body"**——北面是水和绿色，南面是沙和赭红。Ishkar河连接它们——从basin的湿润世界穿过石头，流向沙漠。这条河同时属于两个世界。

**Milo的反应**："That's... that's not a canyon. That's a wound."（或类似的——他第一次见到planetary-scale geology的teenager反应）

**Anna的反应**：沉默。手稳在操纵杆上。她之前飞过这里（或至少看过map），但每次看到还是……她深呼一口气。不评论。继续飞。

**tension level: 0/10 danger, 10/10 awe.** 这是Arc 1的visual climax。

### Phase 4：下降 → Riverbend初见（~500-1,000字）

Anna开始下降进入峡谷。从Grand Canyon上缘（3km高度）向skyport level（300-400m）降低。

**下降过程中的视觉收焦：**

1. 峡谷壁的地质纹理变得清晰——basalt/shale交替层，每一层都有不同的颜色和质感
2. 看到Ishkar河弯——河在这里转了一个大弯。内弯侧的冲积阶地逐渐从texture变成structure
3. **水渠的发现**：峡谷壁面上有一条条闪光的细线。Anna一开始可能以为是裂缝/渗水。Milo发现"那些会动"——是水在人工渠道里流。**从空中看水渠系统 = Riverbend的第一个worldbuilding clue。**
4. **Smoking racks的烟**：河岸平地上成排的棕色木架结构。一缕缕淡烟从那里升起，被峡谷气流拉成水平的薄纱。
5. 城镇本身：阶梯式建筑从河岸平地向峡谷壁攀升。不是chaos——是layered的、organic但有秩序的。每一级阶地的屋顶是下一级的walkway。石阶蜿蜒。

**Milo的台词机会**（worldbuilding delivery）："他们把壁面的渗水全导进渠道了……看，每一级落差都有水车。" 技术boy认出工程逻辑。

**EP13结尾 = EP14开头的bridge：** Anna对准skyport，开始final approach。"Milo. Descent checklist." 切。
