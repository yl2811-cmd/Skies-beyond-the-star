## Arc 4 · Axiom Scar·红夹克引爆·地基补建·同步（Ep56-69）

### 叙事功能

Act 1的终章。做五件事：
1. Anna进入Federation物理遗产（Axiom死船）并带回Kepler Core
2. 红夹克框架在Axiom内部引爆——从"被穿上"到"选择穿上"
3. Kassia的confirmation-seeking在三个递进场景中带回越来越difficult的non-confirmation（Whispering Flats→Axiom→触墙），全部通过Anna可见的external evidence呈现
4. Anna-Lia关系从Ep63重逢出发，通过四集日常地基补建（Ep64-67），让Lia的感知能力和Anna的transmission行为在proximity中完成从"观察者→参与者"的质变
5. Kepler Core与Crystal同步，投影唤醒人类文明碎片——"听"的能力抵达终极边界

### 情感曲线

起锚的约定（presume未来里对方还在）→ 南行摩擦（Kassia在和自己的手较劲）→ 死船深处（三击collision）→ 北归同步的手（correction angle缩小）→ 重逢的"下次帮你听" → 工坊Core入 → 地基补建的四集日常 → 生日后黎明的同步 → Kassia触墙 → Act 1终章。

### Worldbuilding territory

Basin → 绿带河谷 → 西南海岸 → Orun海角翻越（The Cape整个Orun最低翻越点）→ Great Ochre Desert + 铁氧化物沙漠 → Whispering Flats结晶平原（triboluminescence蓝绿色光波）→ Axiom Scar debris field + 500年死船内部 → 返程Basin。

### 不可移动锚点

**Ep56 — 起锚·绞盘约定（~6-8K字）**
方向：Anna确认Axiom科学舱大概率有Kepler Core。芯片在工坊里微光仍在——像等着被接上的心脏起搏器。决定亲自南行打捞。
编队：Anna主驾（中型运输艇）+ Kassia second pilot（同船）+ Milo机械师（运输艇）。Razor绑在运输艇甲板上带着——沙漠段不适合skiff独立飞行，但到了Axiom附近可能需要短程侦察。
**★ Lia送别（核心beat）：** 不是沉重的告别。Lia在Basin。Healer调过一次剂量。这个春天她咳得比去年少一些。也可能是因为春天空气好。Anna没问。Lia没提。出发前清晨。Anna在码头做飞艇最后检查。Lia来了——不是送行，是来看。手指沿缆绳走。
> "这根缆绳比我以为的粗。在手册里它是一个数字。现在我知道它托多重的东西。"（然后）"回来的时候你帮我看看棘轮。间距好像不对。"
**约定**，不是告别。Subtext：我会回来，你在这里，我们有事一起做。
; Reader carry着Ep43 permanence的记忆。"回来的时候"这两个字的weight——
; 一个身体precariousness永远不会走的孩子在说"我还在这里。你去吧。"
; 和Edmund在倒计时里做红夹克的结构echo——不让阴影污染这个moment。

; **四谎register贯穿：** 没有"她好了"的clean signal。"可能是因为春天空气好"让improvement永远是uncertain的。
**Kassia departure beat：** Kassia在甲板上检查绑固的Razor——确认skiff的固定栓和遮盖帆布。她看到了Anna-Lia exchange——不评论。我们记得Arc 3里"你看那个孩子的方式不对"。现在Anna看Lia的方式**变了**——不再是"不对"的那种.

Curiosity不是hope。但真正什么都不在乎的人不会想去"看看"。这是confirmation-seeking——Skyleaf给了三个partial，她需要一个更大的clean confirmation来repair。Federation死了500年，没有比这更彻底的entropy victory。她去"亲眼看看"不是因为好奇——是因为她的framework需要field validation。

**Ep57 — 南行·风中摩擦（~8K字）**
方向：Basin→绿带河谷→西南海岸→Orun海角翻越→Great Ochre Desert边缘。
**地貌转换三段：**
**绿带**：Ishkar支流灌溉的温带林地。Quiver-Leaf颤叶林银色闪烁，Sentinel Pine在山脊站成深色剪影。湿润空气，草和泥味。飞行平稳。Archeon最"地球"的风景。
**海角接近（The Wash）**：海岸线附近空气变咸变湿。Orun的escarpment在这里已不是3km的黑墙——而是一道800m的海蚀崖，底部被浪花打出拱门和海蚀柱。整面崖壁被浓重海雾包裹——从远处看不是黑色，是白色。海蚀台地层层叠叠从海面爬到崖顶，上面长满被海风修剪成匍匐姿态的矮树和厚苔藓。海风updraft强劲且稳定——"光滑电梯"——但需要盲飞。
**翻越脊线→Desert边缘**：利用海风updraft爬升到800-1200m翻过崖顶。穿出雾层的那一刻——阳光。回头看北面是白色雾海，前方是Orun高原面的赭红延伸。继续向东南飞越高原缓降面，高原逐渐下沉、颜色从basalt黑变成沙石赭红、植被消失——Great Ochre Desert出现。
**★ Kassia地形熟悉度种子（方案C·空频段发芽）：** 进入高原缓降面后，一股热柱从赭红地表冲上来。Anna准备绕。Kassia没等Anna反应——手已经在调航线，绕的角度和timing非常准。不是反应快——是**她知道这个热柱在哪里**。
下一个热柱区，同样。Kassia对这段地形的wind pattern有不属于航图的熟悉度——热柱的周期、间距、强度变化。航图上这段路只标了"thermal activity moderate"。Kassia的手知道的比航图多。
Anna可能注意到了——一个只飞Basin主干线的pilot不会有Ochre高原面的肌肉记忆。也可能没注意到——Kassia的hand quality本来就好到让任何correction看起来都是临场判断。
; 回扣Ep38空频段：一个空的、没人在用的频段 + 对Ochre南线地形的body-level熟悉度。
; Reader到这里有了两个data point：她在听一个消失的频段 + 她来过这片沙漠。
; 不需要说出来。Ep58她在这片沙漠上空看着光河睡不着的时候，reader自己会连。
**Anna-Kassia飞行摩擦（核心beat）：** 海雾段中Kassia失去视觉参照。她的精确度依赖视觉——失去视觉本能想加速脱离。Anna减速，让海风updraft替她们做功。
; [感知演化锚点·proprioception·感知连续体极端·Ep44交叉点的mirror]
; ★ Anna在海雾里闭上眼。听不到崖壁（雾把回声吃了）。看不到地面。
; 但她能感觉到updraft——不是通过耳朵，是通过控制杆的微震和自己身体的重力变化。
; 这是比"听"更底层的感知——本体感觉（proprioception）。
; 风筝→滑翔架→skiff→飞艇的连续体在这里走到极端：
; 所有外部感知通道都被环境条件削弱，只剩身体本身作为最后的仪器。
; 不是新能力——风筝线在手里的张力就是proprioception的第一个教室（童年§2①）。
; 但在海雾里，所有后来学会的东西（听、看、闻）都退场了，最原始的那一层重新浮上来。
; ★ 和Kassia的对比跟Ep44高空完全相反：
; Ep44 thin air——Anna失去听觉→降级，Kassia靠仪器→stable。
; Ep57 sea fog——Kassia失去视觉→panic→想加速，Anna失去所有外部通道→退到body→calm。
; 不是"Anna更厉害"——是两种方法论在不同extreme condition下的不同failure mode。
; Kassia依赖external instrument（包括眼睛）。Anna最底层是body itself。
; 呼应：童年§2①风筝（proprioception origin）→ Ep44（Anna降级/Kassia stable）→ 此处（Kassia降级/Anna stable）
> Anna："你的修正角够漂亮。但这艘船不是Razor。三度侧倾，货舱固定栓会被拉到极限。"
Kassia沉默，把手从操纵杆收回——不是被说服，是Anna在物理层面是对的。
**★ Kassia micro-experiment（不correction beat）：** 穿出雾层后。高原面上dry air turbulence。Kassia的手**没动**。这次她等了——让Anna handle。Anna handle了。飞艇稳了。不需要Kassia的提前correction。风自己回来了。Kassia什么都没说。
; 她在测试"不控制是否等于死"。答案是不等于。这个data point被存档了。

**Ep58 — Whispering Flats · Kassia的夜（~8K字）**
方向：结晶平原triboluminescence夜间光河。
**★ Kassia中间环（日间沙漠段·Ep57 not-correction的回声）：** Kassia做primary。一股crosswind——不大，Anna如果在primary会等它过。Kassia的手抬了——要做correction——**停了半秒**。Ep57的记忆在手里。然后旧习惯赢了。她还是做了correction。比平时轻一点——但她做了。风自己也在回来。Kassia的correction和风的自然修正叠在一起——over-correct了一点点，飞艇抖了一下。
Anna感觉到了那个抖。什么都没说。
; 中间环设计：Kassia试过不修正→好的。现在修正→反而worse。
; Ep60绞盘graduated tension有更厚的地基。
**★ Whispering Flats夜间：** 地表是结晶盐壳。夜间风摩擦结晶面时晶体微破裂释放电荷，产生triboluminescence——**蓝绿色光波随风移动**。光随风变——南风起，南边先亮，光波北扩；风转光也转。像大地在呼吸，呼吸的痕迹是光。
; [感知演化锚点·蓝色风信花#5·蓝绿色在赭红里]
; 蓝绿色光波在沙漠夜里扩散开。Anna从飞艇上看到地面上的蓝色ripple。
; 这个蓝和风信花的蓝不一样——是物理的、冷的、无机的。
; 但它在沙漠的赭红里出现蓝色——和风信花在basalt灰壁上出现蓝色——
; 是同一种"不该在那里但在那里"的存在方式。
; 一句话landscape描写。不需要Anna parse。reader的蓝色计数器又响了一下。
; 呼应：童年1.4→Ep25→Ep27→Ep41（蓝眼睛）→此处→Ep62（payoff）
头顶紫色和玫瑰色星云光带。脚下蓝绿摩擦荧光。上下两片冷光之间，飞艇灯火和营火是唯一暖色。风磨结晶面发出持续极低频嗡鸣——"Whispering Flats"是literal description。
**★ Kassia的夜（Anna可见部分）：** 夜间。营地。Anna在某个moment醒了——风或冷。出去。Razor旁边有一盏油灯还亮着。Kassia靠在skiff的骨架上——Anna能看到轮廓，看不到表情。头顶星云光带、脚下Whispering Flats的蓝绿摩擦荧光——上下两片冷光之间，Razor旁那盏油灯是唯一的人造暖光。
Anna没走过去。回去睡了。
第二天Kassia比平时更早起。茶壶底的残渣比一壶多——她喝了不止一壶。说话语速正常。什么都没说。
; writing guide（不进prose）：Kassia的interior是——Whispering Flats是美的。她不否认美。她否认的是美的意义。宇宙不需要目击者就能创造奇迹，所以目击者的存在无关紧要。但她没有离开。看了很久。真正什么都不在乎的人会去睡觉。Kassia睡不着。
; Anna看到的只有"灯整夜没灭"和"茶喝得更凶"。reader从Ep38空频段+Ep57地形熟悉度+这个夜晚自己拼：这个人在这片沙漠上空有她自己的理由睡不着。

**Ep59 — Axiom Scar · 进入死船（~8-10K字）**
方向：远处地平线暗色线逐渐变粗——**百公里长伤疤**切穿赭红沙漠。
**Axiom主体：** Scar终点。半埋沙中——约600米长，像死鲸。500年风沙磨去Federation标识，某角度光线下才见浅蚀刻痕迹。
Anna和Kassia进入（Milo留外面做外部支援监控）。**死寂走廊：** 500年密封，苦涩化学味（密封胶和隔热材料降解）。灯不亮。油灯和反光镜。光在走廊弹射。墙上Federation标准化指示——舱室编号、安全标记、**分区色带**。字体清晰，颜色已褪。
**★ Kassia灯移开（Anna可见）：** 转角。Anna在前面。Kassia举灯照到墙面——500年前船员的记号笔字迹。Kassia**把灯移开了**。继续走。
Anna看到了这个动作。没有评论。不知道Kassia在回避什么。走在后面。
; writing guide（不进prose）：真正什么都不在乎的人不会需要回避。她在保护自己不被某种东西刺到。从Ep58被动睡不着到Ep59主动把灯移开——从"泄露"到"回避"的递进。

**Ep60 — 竖井·她的手没有离开绞盘（~8-10K字）**
方向：拆卸Core→结构位移→被困→垂直货运竖井攀爬。
**拆卸Core：** 隔震基座有紧急手动释放。Anna趴基座下找到把手。握感——光滑、精确、没有锤痕。指尖传来25世纪制造工艺触感。**Federation从传说变成物理实在。** Core解放，约40公斤。
**结构位移：** 拆下Core改变配重分布。支撑梁滑落堵住来路。
**废弃垂直货运竖井**：约60米。Anna在底部绑好Core，Kassia在上面绞。中途Core被卡在竖井内壁突出结构上。Anna沿梯下到卡住位置——单手抓梯档、另一手推Core越过障碍。
**★ Kassia的手：** 缆绳突然松弛了——Anna推过了障碍。然后重新绷紧。
在缆绳重新绷紧的瞬间，Kassia的手做了一个非常精确的制动——不是猛拉（会拽断或拉伤Anna），是**graduated tension**。恰好足够。同一种0.3毫米的精确度——在校准环上是armor，在绞盘上**保护了另一个人**。技能是中性的，context决定它服务于什么。Kassia自己不会notice这个shift。但手知道。
在等待的那几十秒里——缆绳松弛、不知道下面发生什么的那几十秒里——**Kassia的手没有离开绞盘。**
一个nihilist在死船竖井顶部，为一个她声称"doesn't matter"的任务绞着缆绳，等着一个她声称不在乎的人从黑暗里爬上来。
; 不给Kassia interiority narration。只给行为。让我们自己读。
Core上来了。Anna上来了。破裂外壳裂缝——蓝天和沙漠热浪涌进来。Anna放下Core时膝盖发软。Kassia看了Anna一眼。没说话。走向Razor。

**Ep61 — 红夹克引爆 · Kassia独自回船（~12-15K字）★★★ Arc 4核心**
方向：物理触发不是心理描述。遵循全书记忆系统规则：body memory通过具象感官匹配activate。
白天在Axiom走廊里，Anna注意到了舱壁上的**分区色带**——蓝色工程区、绿色生命支持区、橙色科学区。500年前的设计：走进一个颜色区域，所有人知道你是谁、你来做什么。**颜色在你到之前就告诉所有人你是什么人。**
营地晚上。Anna在Core旁整理设备。手指无意中又碰到从Axiom带出来的一块色带残片的edge。触感——Federation级精确切割的金属色带edge，和Archeon手工金属的粗糙完全不同。
