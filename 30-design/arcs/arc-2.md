## Arc 2 · 回乡与Edmund记忆三部曲（Ep20-28）

### 叙事功能

Anna回到Atheria，我们第一次peel开Edmund层，然后送Milo走（Ep25）。Arc 2全程的组织原则：**先让我们爱上Edmund这个人，再看他给Anna留下了什么，最后一起失去他。** 顺序不能换——如果Ep22就展示Edmund的教学，我们会把他归类为"mentor"（功能性角色）。必须先看到他搞不定面粉、被羊甩到泥地里、忘了吃烤黄头串。

### 情感曲线

Atheria归属感 → 童年纯joy爱上Edmund → 日常感官触发记忆碎片看到继承 → 失去 → 天空独自回来 → 走梭人完整工作日的成年arc。每集4-10K字。

### 两集用不同的记忆组织原则解决年龄跳跃

**Ep22 = 时间驱动。** 正蒙太奇从3岁往上长到9岁。Edmund从一种感觉凝固成一个活人。年龄必须单调递增。

**Ep23 = 感官驱动。** Anna回到present-time鹰巢工坊。手碰工具→触发M-06河石（6岁）；引擎声→触发M-14和弦（9岁）；站在平台上往外看→触发M-05分层（5岁）。感官触发天然不遵守时间顺序。年龄跳跃在这个模式下是feature不是bug。

两集切换点：Ep22结束在雨天工坊（最安静最深的沉浸）。Ep23开头Anna从记忆中醒来回到19岁的鹰巢——present-time重新接管叙事。

### Worldbuilding territory

Atheria三层结构完整展示（谷底人口重心+崖壁产业+崖顶交通门户+在建崖壁中段sky pier）→ Ep22正蒙太奇回溯童年（鹰巢工坊+峡谷+河）→ Ep26a航线巡回（Millhaven磨坊+Basin平原outpost+海岸过渡）→ Ep26b-27 Iris Hollow（海岸平原+面粉消息链+6年关系old man）。



### 不可移动锚点

*### Ep20：回乡·落地·社区温度

Anna和Milo驾Hauler回Atheria峡谷。

**★ 进场——从高空到峡谷口（reader第一次看到Atheria）：**

这是reader第一次看到Atheria。reader在之前8集里只知道Anna是一个好pilot，不知道她从哪里来。这个进场scene的每一个视觉/听觉元素都是reader对Anna故乡的first impression——也是后续所有"回家"场景的baseline记忆。seed密度和多样性至关重要。

**高空段（云层之上→峡谷口）：**

Hauler从大地深处方向飞来。高度在云层上方。下面的地形是赭褐色和绿色拼接的——田野、颤叶林银色带状闪烁、偶尔一个小outpost（几栋建筑+一座太阳能碟+草地锚桩上系着的小飞艇）。但不是一张均匀的毯子——下面起伏的丘陵脊线投出明暗交替的条纹、几座basalt台地从绿色海面上孤立隆起、浅切河谷把大地切成不同高度的层次、远处一片湿地反射出一条银色弧线。然后地形开始碎裂——大地的边缘突然跌落，峡谷像一条黑色的裂缝切开下沉前的最后一块台地。Ishkar河北支从高处看是一根深绿色蜿蜒的线。

跨过云层的时候，两三架skiff从崖壁顶部滑下——单人小艇，两三米宽，半裸露的open cockpit，一个harness固定飞行员，下方挂着小货篮。它们骑峡谷壁面热力差产生的下降气流，没有引擎的持续低频，只有风穿过金属框架的高频哨声和辅助涡轮偶尔kick in的短促呼噜。有些歪歪扭扭的，明显是某个中年人在从崖壁顶下到谷底去买东西。像大型鸟。Anna在Hauler的高度上看着它们从下方掠过——这是Atheria的"自行车"。日常的、随处可见的。Ep25比赛里这些skiff会以竞速形态出现（去掉货篮、翼面调为速度优化），但此刻reader先看到它们的日常版本。

**峡谷入口段——声音layer-by-layer到达：**

Hauler开始下降。进入谷地上空——两侧缓坡渐渐展开在下方，光线变暖。声音不是一下子全到的——它是一层一层叠加的。

先是风声（高空，干净，只有风和引擎）。然后坡面出现在两侧——声音开始有反射了，但不是硬墙的清晰回弹，是沿缓坡散射的温暖混响。然后**低频先到**：升降笼绞盘的aggregate hum、坡面中段工坊带的aggregate（锤击、焊接混在一起的连续底噪，从basalt阶地里传出）、风车叶片转动的低沉呼吸。然后**中频**：单独的锤击声能分辨了、某个工坊的金属切割声、之字路上的脚步和thorn-goat蹄声。然后**高频**：管铃叮当、人声的碎片、Ishkar河水声从谷底升上来。

到达skyport高度的时候——所有声部同时在。Anna不做任何辨认动作。她的身体自动register：**这是对的。这是家。**

**视觉——从空中俯瞰谷地的基础设施分层（自上而下）：**

; 设计原则：每一层用不同的感知通道和不同的基础设施类型。
; 绝对不出现"风塔频率"或任何单一设施的诊断。这是homecoming，不是inspection。

**坡顶plateau**——skyport平台，停泊架上系着一两艘小飞艇。平台边缘一排信号灯桩——木杆顶端挂油灯，白天不亮，但灯罩金属在阳光里一闪一闪。对面坡顶也有一片plateau——钙钛矿太阳能板沿天然台阶排列，深蓝紫色的矩形面板像鳞片贴着斜面，有些旧的表面发花，有些新的很干净。

**坡面上层**——风力涡轮叶片在远处转（视觉背景层，不做焦点）。一两个气象/通讯观测桩——细长金属杆，顶端风向标。

**坡面中层（产业带）**——沿basalt阶地排列的工坊和住宅，层叠如阶梯。有些窗里有火光闪动。之字路蜿蜒在建筑和阶地之间，路上有人在走——从这个高度看像是在斜面上缓慢移动的点。偶尔一段蒸汽管网排汽口吐出白色小羽毛在风里飘散。有些段落比其他段落陡——Anna能认出鹰巢所在的那段近垂直短崖面，一小片平台伸在那里，从上面看很不起眼。

**空桥**——坡面中段少数窄处有basalt fiber+铆接铜板建的空桥跨越同侧崖面缺口（翠绿色铜锈+墨绿桁架），连接不同高度的工坊阶地。一座空桥上有一队刺角羊在过桥——有人在后面赶。

**升降笼**——两三个金属箱子在不同高度沿钢缆移动。有些运货（外面绑着布包），有些运人（侧面开放，能看到里面站的人）。上行慢、下行快。

**谷底（从高处俯瞰）**——2公里宽的开阔盆地。Ishkar河蜿蜒穿过——三四十米宽，墨绿偏蓝，几座石拱桥和木桥横跨河面。河弯处冲积平地上是最密集的建筑群。银色灌溉渠从河里分出来像血管，分支进入绿色田垄。一条窄轨轨道沿河岸延伸——偶尔能看到一辆小型氢电池平板车在上面嗡嗡地跑，拉着几箱货或坐着几个人。**风车散布在田野之间——从高处看是缓慢转动的圆，basalt fiber叶片墨绿色。** 小木屋群散布在步道和渠边——坡屋顶、有些屋顶长了草、建筑之间拉着晾晒架——布料、草药、肉干——水平的彩色线条。鸡群在院子里走来走去——从高处看是棕红色的小点在缓慢移动。Common house旁边那棵老白蜡树的深色树冠。有炊烟从几个方向升起来——被微风吹成弧线，在低空形成一层金色薄雾。

**★ 飞艇影子beat：** Hauler下降过程中，它的影子在谷底田野上滑动。一个弯腰干活的人抬头看了一眼，然后继续弯腰。Anna从高处看到了这个人看了她一眼然后不看了的全过程。**她到家了。** 大人不看飞艇——习惯了。只有孩子还抬头。

; [设计备注：这个scene的每一个seed都是紫罗兰映射——升降笼在动=有人上班，
; 空桥上有羊=有人运输，风车在转=风在、渠里有水=田里东西在活，
; 晾晒架上有东西=收成不错，skiff从坡面skyport滑下=有人在送午饭。
; Anna不需要说出任何一条。reader自己感受到：这个地方在运转。所有系统都在。
; 这是一个活着的、忙碌的、被山坡抱住的家。]

**★ Kassia intro（skyport · 30秒 · POV transfer）：**

Anna和Milo在skyport完成Hauler交接。旁边一架Razor正在被检查引擎。从那个方向传来一个声音——不是对Anna说的。一个黑发pilot在跟dock mechanic说话，语调直接、不客气。她在说某个校准环的偏心量——"0.3毫米。你觉得这在公差内？你用什么量的？"dock mechanic在辩解。她不听辩解——在等一个数字。
POV在这几秒里短暂落在Kassia身上：她的手指在校准环内壁滑了一圈，指甲刮过金属面的声音很轻。手指停在一个位置。"这里。感觉到了吗。"dock mechanic弯腰看了一眼。
dock mechanic开始重新校准。Kassia退后一步，从口袋里掏出什么——一块干果或一截肉条——嚼了一口。旁边一个学徒路过，看了她的Razor一眼。Kassia对那个学徒："看够了就帮忙递那个扳手。"学徒递了。Kassia没说谢谢，嘴角动了一下。
Anna经过。"Kassia。"
Kassia抬头看了一眼。"Freedman。"回到引擎。

**设计约束**：建立professional精确度+声音sharp+两人认识但不close。不给nihilism暗示。**同时建立日常两面性baseline——sharp和warm在同一个30秒frame里都出现。** 她不是一个只有precision的人，第一个frame就让reader看到她会吃零食、会对学徒嘴角动一下。

**视觉sequence——落地后沿坡面下降（ground-level体验·角色依次出场）：** 上面的进场是从空中看到的Atheria。落地后换频道——从overview到body-level。
**★ Hagen（第一张脸）：** Skyport平台上。老pilot已经在那里了——不是来接的，是碰巧在修什么。看到Hauler降落，直起腰。不急不慢走过来。和Anna交接货物manifest的时候检查了一下舷板接缝——手指沿金属边缘滑过去，和Anna小时候看Edmund检查的方式一样。"有点松了。下次在Ironvein买铜铆钉的时候帮我带两盒。"第一句话是工作。第二句话不说。但他站在那里的方式就是第二句话。
沿之字路下谷或坐升降笼——Anna从skyport下到谷地的过程中，在不同高度层遇到不同的人。空桥层学徒问骨架赛；
**★ Joren（工坊层）：** 经过鹰巢工坊所在的坡面中段——那段近垂直短崖面的顶缘。工坊门开着。Joren在里面——不是在等Anna，是在做自己的事。一个solder joint。Anna进门的时候他没抬头。"你回来了。""嗯。""Ishkar北渠第三节阀门上周换了。旧的在角落。"两句话把维护状态交接完了。Joren维护过Edmund走后的鹰巢。他不是替代者——是在空位旁边默默补上的人。Anna拿起旧阀门看了一眼修痕。Joren的手法和Edmund不一样——更直接、更functional、没有Edmund那种不影响功能的小花纹。但东西work。放下。继续往谷底走。
之字路往下走——每一段温度和气味都在变。中层工坊带的金属味和碳火味渐退，低段渐渐变成草味和水渠声。路边老人说"又瘦了"。升降笼的绞盘声从不远处传来。管铃在头顶叮当。

**盆地横穿（约两公里）：** 升降笼出来，脚踏实地。盆地底部是开阔的——和垂直段的窄通道完全不同的scale。午后阳光斜打进峡谷，只照到盆地西半边，东半边在崖壁阴影里。Ishkar河从盆地中间穿过，石拱桥上有人走。河边水车在转——木质的，叶片上挂着水滴在阳光里闪。桥那头是温室群，玻璃和旧solar panel拼的屋顶反射出不规则光斑。田垄之间有人弯腰干活。远处一排低矮住宅贴着对岸崖壁底部，屋顶上管铃在风里叮叮当当。

Anna走在压实的泥土路上。红夹克在午后的光里很亮。身体在自动校准——水车节奏比走之前快了，水位高了？崖壁中段新一段sky pier的影子比上次更长了——又往谷里伸了几米。

路边矮石墙上坐着几个孩子。一个在踢墙根的草。两个在争什么——手里各攥着半截铜管，指甲缝里有铁锈，从崖壁工坊出来的。还有一个坐在最边上，腿悬着，没在跑——手里拿着一片叶子在看。不是本地品种——从河边飘上来的，形状和峡谷里常见的不一样。Anna走过的时候那个女孩抬头看了一眼红夹克。然后低头回去看叶子。Anna继续走。步速没变。

**★ 灰雀迁飞（Anna到达Atheria的第一个感官事件）：** Atheria换季。崖壁灰雀群起飞，几千只在峡谷里形成流动云团，声音叠加成有节律的嗡鸣。Atheria人的换季信号——"灰雀走了，凉季快来。"飞行员看鸟群的方式不同：看流体动力学，看它们怎么利用上升气流、在崖壁转角变换队形。Anna指给Milo看鸟群在凸角处的分流汇合——"它们不穿过乱流，它们绕开。和我们做的一样。"灰雀迁飞给reader对Atheria的第一印象是"活的、有呼吸的、会换季的地方"。

过河——穿过温室区域（玻璃棚内绿色植物在午后光里发亮，空气里有泥土和水的味道）——到达对岸崖壁底部——往上——Common house。Miriam抬头看一眼："饿了吧。锅里有。"Anna吃了两口，Miriam从另一个房间走出来，在Anna身后站了一秒，手放在Anna肩上，然后拿走空碗去洗。不说话。

### Ep21：日常·Guardian身份·种子·记忆门槛

Anna的guardian日常——检查水力系统、评估学徒进度。Atheria日常worldbuilding：早晨雾气（水汽从Ishkar河面升起，在峡谷壁上凝结，形成一天的timing marker）；午间峡谷风转向（壁面温差导致气流reversal）；傍晚Ishkar河水位变化作为"时钟"（上游冰雪融水日周期）。鹰巢工坊内部：多层工作台、悬挂工具架、嵌入岩壁的管道系统、从崖壁开口射入的斜光。声景：锤击声在峡谷壁之间的回响pattern。Edmund的absence通过环境暗示——工坊里的空位、工具架上按Edmund习惯排列的工具（Anna从没改过）、别人说话时绕过一个名字。

**种子播种：** Anna和Milo检修压力系统，Milo步骤顺序不对（先释压再检查），Anna的手直接按住Milo手腕："先检查，再释压。永远是这个顺序。"我们只看到"严格的师傅"。真实层：这正是Lia事故那天13岁的Anna没有做到的事。Anna的guardian工作覆盖valley大部分区域，但有一条路线始终没走。Milo提到"那边的水泵也该检查了"，Anna只说"那个在别人的轮次里"。我们只看到"正常的工作分区"。真实层：那是Lia事故现场。

**修渠日（白天·公共劳动传统）：** Atheria水渠系统定期集体维护——清淤、换垫片、检查水轮。全谷参与，是社区活动不是punishment。各层居民沿水渠从上到下排成接力线，一边干活一边传八卦。有人唱工作歌配合铲子节奏。修渠日是信息交换的窗口——飞艇带来的消息被二次传播。Anna的角色从"本地居民"变成"走梭人带消息的人"——被各种人拉住问南边怎么样。Milo第一次参与，结果被一个八岁本地小孩两秒演示了正确铲淤姿势，还翻了白眼。尊严和幽默同时被激活。

; ★ 功能：为Ep42-43 Lia翻译三秒停顿的情感能量做前置铺垫。
; 如果没有这个"没人接住"的空beat，Lia"接住了"的weight不够——
; Lia是六年来第一个从外部接住Anna感知的人。
; 呼应：M-06（Edmund共享delight）→ 此处（共享消失）→ Ep42-43（Lia接住）

| 表层 | mechanic观察年轻人干活 |
| 误读层 | 想起自己当年跟Edmund学艺的nostalgia |
| 真实层 | 那个孩子13岁——Anna犯下Lia事故时的年龄。她在看曾经的自己 |

**段3：海岸接近（~800-1,000字，飞行中，不靠港）**

不是一个settlement stop——是一段飞行中的感官过渡。

离开平原outpost后继续西飞。地形开始变——平原的金色渐渐掺入灰绿，草变短了，灌木多了。

湿度上升。Anna的皮肤先感觉到——空气里多了一层她说不出名字的东西。盐。不是咸——是一种矿物质的存在感。

光线变了。内陆的光是硬的、干的、有棱角的。接近海岸时光变柔了——海雾把阳光过滤成灰白和珍珠色。

声音开始变。引擎声在这种空气里传播的方式不同——更闷、更短。远方隐约有一种低频的存在——不是风，是海。

Anna的body在做recalibration——同一个过程，她14岁第一次来时做了半天，现在三分钟完成。不以闪回形式出现，但读者在Ep27看到backstory·二"海岸三天"时会认出这个过程。

**段4：Iris Hollow外围（~300-500字，过渡到Ep26b）**

### Ep26b — Sky Weaving · 到达Iris Hollow（4,000-10,000字）

**穿过云层。** Anna从海雾/低云层中穿出来。视野打开。

**Sky Weaving编队**（概念图Image 8）：几架Iris的skiff飞到Anna的阵风号旁。尾部拖染色长布带，在田野上空编织交叉彩色轨迹。不是比赛，是协作。

Sky Weaving的physical sensation：Anna习惯了独自飞行（这几天solo更强化了这个习惯），编队要求她感知五六架skiff的相对位置和速度——不是用仪表而是用周边视觉和气流扰动。旁边的skiff经过时产生尾流，她要学着"读"这些尾流来维持编队间距。从独奏变成合奏——和她"听"机器的能力是同一个认知族群的不同分支。

帆的echo线：六岁独自放风筝 → Ep25独自竞速 → Ep26b协作编队。三种"和风一起玩"的方式，progressively更social。

**Iris Hollow展开worldbuilding。** 到达后的地面体验——这是reader第一次进入Iris Hollow。沿海丘陵settlement。没有崖壁可附着——空港利用丘陵坡面做terrace，更水平、更spread out。小型飞艇直接系在草坡的地面锚桩上，像一群在草地上吃草的巨大水牛。海风带着盐味。周围林地资源丰富，建筑用更多木材，气质比Frontier更pastoral、更开阔。

**到达后的走梭人moment：** Anna把Millhaven面粉消息带到面粉姑姑——"今年粉粗了半号，发面时间多给一刻。"一条信息链从磨坊主嘴里→Anna→面粉姑姑的手里完成了全程。The Loom的活体神经。

### Ep27 — Iris Hollow日常 · 老人 · 声学失灵 · 故事夜（4,000-10,000字）

**★ 老人的感知方向观察（六年关系 + Edmund网hidden node）**

老人锚定在Iris Hollow本地。

; [P-10 ★★ OVERRIDE P-08.1 §C]
; 从"手生了"/mechanic identity poke改为感知方向变化观察。
; Anna从来不是mechanic——她是listener。老人看到的变化是注意力方向的shift。

**六年context——保留：** Anna从14岁开始独自飞行后，一年经过Iris Hollow约2-3次。有时专程来修东西，有时是路过时"顺便"停一下。老人每次都有"不急但可以修"的活给她——不是制造问题，是把一些settlement总有的非紧急维护攒着留给她。

Anna在mental map里给这个人的slot是："Iris Hollow那个总有东西需要看看的老头"。不是"爸爸托付的人"。六年下来，这是一个comfortable professional relationship。

; [P-08.1 §A 网的运作方式——保留]
; Anna蹲下来修东西的时候，她在吃那家人的饭，睡在他们腾出来的房间里。
; 她不是在"被照顾"。她在工作。尊严完好。但同时也在被人的温度包着。
; Care被藏在commission里。网的节点不知道自己是网的节点。

**Scene context：** Anna到Iris Hollow，是以**pilot身份**来的——接了个飞行任务。不是来修东西的。

老人认出她。六年的regular。"刚好有个东西想让你看看。"

Anna可能本来有别的安排。但老人说了——而且在Loom的逻辑里，一个你认识六年的人说"帮我看看"，你不会拒绝。Anna接了。

**★ 感知方向shift moment（替代旧版"三层叠合identity poke"）：**

Anna蹲下来修老人给她的东西。她修的时候competent——能力还在。但老人注意到了一个变化：

**老人记得的10岁Anna：** 跟Edmund来的时候，那个孩子会蹲在什么东西旁边听。不是修——是听。Edmund在旁边干活，她趴在管道上、贴在墙上、把耳朵凑到什么震动的东西旁边。她的注意力全在**近处的东西**上——机器的频率、零件的声音、结构的震动。整个人蜷在机器旁边。

**现在的Anna（19岁，pilot）：** 她来了但老人注意到她的注意力不一样了。她会在干活的间隙抬头看风向。她的耳朵在追**远处的东西**——不是管道的震动，是气流过屋檐的声音、远处天气变化的低频。她的身体朝向变了——19岁时身体是展开的，像随时准备离开地面。

; [写法约束] 注意力方向的shift通过Anna在修东西过程中的行为传达，不通过narrator解释。
; Anna抬头看天、侧耳听远处的风、身体朝向的展开——这些是可观察的行为，
; 和Ep23-24回忆篇里10岁Anna蜷在机器旁边的画面形成对比。
; 但对比在prose里不explicit——两个画面分在不同episode里，reader自己connect。

**老人的观察——注意力方向的casual remark：**

修东西的过程中或修完后。Anna在干活时抬头看了几次天。老人注意到了。

候选台词方向（写作时择一或融合）：
- "你小时候能在一个水泵旁边蹲一下午。现在你好像总在听外面的什么。"
- 更隐晦的版本：Anna抬头看天，老人说"风要变了？"Anna说"嗯，晚上可能有南风。"老人说"你以前不看天的。"

**Anna的反应：** 轻的。自然的。"现在飞得多嘛。"

; [★ P-10 设计约束——替代旧版"手生了"约束]
; 老人关于注意力方向的观察在第一遍读是transparent的。
; "一个pilot当然关注天气和风向"——完全合理，不carry任何hidden weight。
; 它的devastation只在retroactive reparse中出现（Lia事件展开后）。
; Post-Lia reparse：她的感知从近距离退向远距离不只是因为"飞得多"——
; 是因为近距离接触意味着她能伤人。飞行不只是新的skill domain，
; 是她的整个感知系统从机器/地面逃向天空/气流的方式。
; 老人不知道"以前"和"现在"之间站着一个7岁的孩子和一把飞出去的扳手。
; 但Lia事件展开后reader知道了。

**老人提起Edmund（texture，非payload）：**

修完东西之后（也许是吃饭时，也许是Anna准备走的时候）。自然的conversation beat。

老人看着Anna干活的样子，说了一句："你小时候你爸带你来过。"

Anna在收工具。"他带我去过很多地方。"

轻的。六年关系里这种话说过不止一次。不carry任何special weight。

; 这句话不是Ep61 trigger。它是六年关系的texture——老人和Anna之间有Edmund作为shared history。
; 它的存在让老人scene更有temperature，但不承担远投弹功能。
; 如果写作时觉得这个beat让scene分心（从感知shift上），可以直接删。

Anna的反应：轻描淡写。可能在收工具——hands busy。

**这颗子弹已移除。Ep27老人scene只做一件事：感知方向shift + 六年关系texture。** Edmund网的accumulated warmth在Arc 1-4通过distributed warmth beats实现，不在此scene集中。

**设计约束**：
- 老人关于注意力方向的观察必须是casual tone——不是诊断，不是感慨。
- Anna的反应要快、要平。不是压抑——是真的没觉得这有什么特别。
- 不在这个moment给任何identity shift的内省。子弹打进去了但没炸。
- **老人和Anna的relationship必须有六年的texture。** Prose需要conveying familiarity。
- 老人最小viable characterization：也许他话多。也许他做饭很好。名字、年龄、具体身份待scene writing时确定。建议不是authority figure——越普通越好。

**★ Iris Hollow声学失灵 + 新诊断方式**

**声学特征（Anna的"听"第一次失灵的核心原因）**：

Atheria峡谷是天然的声学放大器——两面崖壁之间的回声让Anna通过sound location定位故障。Iris的平原没有反射面。声音发出去就散了，没有回弹。Anna习惯的"从回声角度判断声源距离"的方法，在这里等于盲了一只耳朵。

风是另一个问题。Atheria的峡谷风是层流的、有方向的（沿峡谷轴线），Anna可以把风声当成背景滤掉。Iris的平原风是湍流的、无方向的，它把所有机械声揉成一团模糊的白噪音。

Anna第一次站在Iris的风力设施群旁边的disorientation moment——闭眼听，听到的不是独立声部，是一锅糊。

**Iris的替代感知方式**：
- 看管壁外侧的凝水pattern（凝水均匀=正常；偏移=堵塞或摩擦）
- 看田间植物叶尖的卷曲方向（灌溉水压不够的地方植物先萎，方向指向故障源）
- 摸土壤的湿度梯度

Iris人不需要"听"——他们的世界是视觉的和触觉的，因为平原上看得远、风太大听不清。

**Anna的认知突破**：Edmund教她的方法不是唯一的方法。不是错的，但是local的。为后面Frontier和Skyleaf的adaptability做铺垫。

**故事夜（收获季传统）**：

和Atheria的"黄昏灯火"形成对比。Atheria的美是垂直的、碎宝石式的（崖壁上点点灯火）。Iris的美是水平的、篝火式的——一张长桌铺在打谷场上，两端各一堆篝火。桌上不是精密工具，是面包、奶酪、谷酒。

讲故事的方式不同：Atheria人讲故事像写logbook（事实、数据、因果链）。Iris人讲故事像唱歌（夸张、重复、有节奏的call-and-response，台下在固定的地方齐声接话）。

**故事夜下半——"天上来的人"（与Ep21 Atheria上半形成文化差异折射）：** 不同的火堆、不同的光线（海岸渔村的咸湿空气 vs Atheria峡谷的干燥矿物气）。不同的故事人。**Lia不在场——Lia留在Atheria。** Iris Hollow故事人讲的是Atwood一代从地球来、被困住的故事。521年的以讹传讹更严重："一个所有人都在睡觉的星球"、"那里的房子会自己长出来"、"他们的天上没有星云"。Atwood ledger被抄了很多遍，有些细节保留了（数字、日期），有些变成了隐喻。Iris Hollow故事人更pragmatic（渔村性格）："我不知道那边是不是真有人在等我们，但我知道明天风向变了之后鱼会往北走。"同一个故事的两种折射：Atheria关注"旧声做了什么"（技术和建设），Iris Hollow关注"天上来的人为什么来"（动机和选择）。两个settlement的identity通过同一个故事浮出来，不需要exposition。

Anna被邀请讲Atheria故事。她用Atheria式的方式讲——精确、干燥——台下安静了一会儿，然后一个老人说"你讲的是事情，不是故事。再来一遍，这次讲给我们听你闻到了什么。" Anna被迫用不同的方式组织记忆——不是因果链，是感官。这个过程是一次emotional processing——她第一次不得不用"味道、温度、光线"来重述Edmund教她的那些时刻。

**阳光莓echo**：Anna在故事夜被要求"用味道和温度讲故事"时，阳光莓的记忆被主动调用——她第一次试着把这个记忆讲给陌生人听。

; [感知演化锚点·蓝色风信花#3·故事夜dialogue嵌入]
; Anna讲阳光莓时提到鹰巢附近的风景——"阳光莓长在朝阳的岩壁高处，旁边裂缝里有一丛蓝色的小花"。
; 这是Anna在讲给陌生人听的时候无意识地把风信花带了出来。她自己不觉得这有什么特别——
; 那就是她家门口的东西。但reader在Ep62回看时会记起来。
; ★ 蓝色在Anna的记忆里是"背景"——不是被注意到的东西，是一直在那里的东西。
; 这恰好就是P-10的定义。半句dialogue。不emphasis。
; 呼应：童年1.4（origin）→ Ep25（空中俯瞰）→ 此处（记忆中的背景色）

; [感知演化锚点·感知表达方式节点]
; 故事夜"用味道和温度讲故事"的moment本身是一个感知演化节点——
; Anna被迫从因果链组织切换到感官组织来重述记忆。感知的表达方式变了，不只是感知本身。
; 这是Edmund轴（因果链/逻辑）和Miriam轴（感官/活物）在表达层面的第一次交叉：
; Anna用Miriam的方式重新组织了Edmund教她的内容。
; 不需要新scene——故事夜本身承载。Writing时标注这层即可。

; [Anna solo的故事夜vulnerability]
; 如果Milo在，Anna可能把"讲故事"这个任务dodge掉（让Milo说两句，自己补充技术细节）。
; Solo时她没有buffer。被要求"讲你闻到了什么"时，她只能自己面对Edmund的记忆。
; 这是Milo离队的叙事功能之一——让Anna在Iris Hollow的emotional exposure无遮挡。
